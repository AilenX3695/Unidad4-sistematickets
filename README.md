# Unidad4-sistematickets

Mini aplicación SPA (HTML/CSS/JS) - Sistema de Tickets.

## Estructura
- index.html
- css/styles.css
- js/app.js

## Requisitos
Navegador moderno (Chrome, Edge, Firefox).

## Ejecutar localmente
1. Abrir `index.html` en el navegador (doble clic) o servir con `Live Server` en VSCode.

## Publicación (GitHub Pages)
Se publica la rama `gh-pages`. URL: `https://AilenX3695.github.io/Unidad4-sistematickets` (después de push).

## Funcionalidades
- Formulario con validación en UI.
- Guardado en localStorage.
- Filtrar por prioridad, buscar por texto, eliminar, marcar leído.
